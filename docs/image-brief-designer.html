Images Required

1. Hero Mockup 1400 × 900 px

Placement: Below hero tagline on homepage (main visual)

Content:

  • iPad (landscape or portrait) showing employee clock-in screen — dark mode, PIN keypad visible, "Kệ" logo top of screen
  • MacBook or iPhone beside it showing owner dashboard — light mode, list of clocked-in staff
  • Subtle purple glow / shadow effect behind devices

Background: White or very light lavender (#F7F4FC)

Style: Clean product mockup, no lifestyle photography

File: hero-mockup.png (PNG with transparent or white bg)

2. Feature — Clock-in Screen 700 × 480 px

Placement: Features section, "Time tracking" card

Content:

  • iPad close-up, dark background (#1a1a2e)
  • 4-digit PIN keypad with dots showing entry in progress
  • Employee name visible (e.g. "Hi, Lisa 👋")

File: feature-clockin.png

3. Feature — Owner Dashboard 700 × 480 px

Placement: Features section, "Time tracking" / "Reports" card

Content:

  • Browser or iPhone showing owner dashboard — light mode
  • List of 3–4 employees, clock-in time visible, green "Clocked in" status
  • Today's date and shift summary visible

File: feature-dashboard.png

4. Feature — Tips Report 700 × 480 px

Placement: Features section, "Tip distribution" card

Content:

  • Tips report screen — light mode
  • Table showing employee name, hours, tip amount (e.g. Lisa · 8h · $42.50)
  • Total tip pool amount at top (e.g. "Total: $180")

File: feature-tips.png

5. Owner vs Employee Split View 1400 × 560 px

Placement: "Two views. One system." section

Content:

  • Left half: owner dashboard on MacBook/browser — light background, white UI
  • Right half: employee clock-in on iPad — dark background (#1a1a2e), purple accents
  • Thin divider line or subtle gradient between the two halves

File: split-owner-employee.png

Style Notes for All Images

RuleDetail
FormatPNG (preferred) or WebP · @2x resolution if possible
Device framesUse realistic device mockups (iPad, MacBook, iPhone) — no cartoon frames
UI contentUse sample business: "ABC Nail Salon" — employee names: Lisa, Anna, Jenny
Data shownAll numbers must be fictional (e.g. $42.50 tip, 8h shift) — no real data
Logo in UIUse Kệ logo mark (small, top-left of app screens)
PhotographyNo lifestyle photos of people — product UI only
ShadowsSoft drop shadow acceptable on device frames
Purple gradientAccents, buttons, highlights = gradient #4B00B2 → #9A29E2
